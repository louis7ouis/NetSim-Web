# NetSim
**Netzwerk Simulator fürs Web — moderner als Filius, kein Java, kein Install.**

NetSim ist ein kostenloser Netzwerk-Simulator direkt im Browser — entwickelt für Schüler und Auszubildende. Einfach den Link öffnen und loslegen. Funktioniert auf Windows, macOS, Linux und iPad.

---

## Features

- 🖱️ **Drag & Drop Topologie** — Geräte platzieren und per Kabel verbinden
- ▶️ **Entwurf & Simulationsmodus** — Wechsle per Klick, sieh Pakete animiert fließen
- 🌐 **Webserver, DNS, DHCP & mehr** — Reale Protokolle, simuliert
- 💻 **Terminal mit ping, traceroute, ipconfig** — Auf jedem Gerät verfügbar
- 💾 **Projekte speichern & laden** — Als JSON exportieren, jederzeit wieder öffnen
- 🎓 **Gemacht für die Schule** — Klar, übersichtlich, ohne Einrichtung nutzbar
- 📱 **Plattformunabhängig** — Browser genügt, kein Install, kein Admin nötig
